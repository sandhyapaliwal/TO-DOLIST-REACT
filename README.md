# Task Manager App

A React-based task manager application that allows users to add, edit, and delete tasks with persistent data storage.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Data persists after page refresh (localStorage)
- Clean and responsive UI

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3

## Getting Started

### Prerequisites
- Node.js installed
- npm installed

### Installation

1. Clone the repository
```bash
git clone https://github.com/sandhyapaliwal/TO-DOLIST-REACT.git
```

2. Navigate to project directory
```bash
cd TO-DOLIST-REACT
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```

5. Open browser and go to `http://localhost:3000`
